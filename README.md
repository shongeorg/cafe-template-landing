# ☕ Café Aroma — Професійний Шаблон Лендінгу

Це сучасний, швидкий та доступний шаблон для створення простих лендінгів (на прикладі кав'ярні). Побудований на чистих технологіях (HTML/CSS/JS) з фокусом на продуктивність та SEO.

## 🚀 Основні особливості

- **100/100 Lighthouse**: Оптимізовано для максимальної швидкості завантаження та SEO.
- **PWA (Progressive Web App)**: Сайт можна встановити як застосунок, він працює офлайн завдяки Google Workbox.
- **Dark Mode**: Вбудована підтримка темної теми зі збереженням вибору користувача.
- **Accessibility (a11y)**: Відповідає стандартам WCAG, підтримує навігацію клавіатурою та скрінрідери.
- **SEO Ready**: Впроваджена семантична верстка, Open Graph теги та структуровані дані Schema.org.
- **BEM Architecture**: Чистий та масштабований CSS за методологією BEM.

## 🛠 Технологічний стек

- **Frontend**: HTML5, CSS3 (змінні, Flexbox, Grid), Vanilla JS.
- **PWA**: Google Workbox.
- **Тестування**: Playwright, Axe-core.
- **Інструменти**: live-server.

## 📦 Як почати

1. **Встановіть залежності**:
   ```bash
   npm install
   ```

2. **Запустіть локальний сервер**:
   ```bash
   npm run dev
   ```
   Сайт буде доступний за адресою `http://127.0.0.1:8081`.

3. **Запустіть тести**:
   ```bash
   npm run test
   ```

## 📂 Структура проекту

- `index.html` — основна розмітка та мета-теги.
- `index.css` — стилі (BEM + темна тема).
- `index.js` — логіка (меню, скрол, перемикач теми).
- `assets/` — оптимізовані зображення.
- `sw.js` — сервіс-воркер для офлайн-режиму.
- `manifest.json` — маніфест PWA застосунку.
- `test/` — автоматизовані тести Playwright.
